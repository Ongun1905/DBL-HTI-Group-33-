Visualization 1: MultiDiGraph

This visualization consists of a Network MultiDiGraph, where each node represents a user's e-mail address and each edge represents a message sent between two users. With the help of interactive tools, you can easily analyze the e-mail traffic of Enron's company.


Interactive tools - Left panel

Select from the left panel the filters that you want to apply for the data, then please click the "Update Graph" button.


You can also hover over nodes in order to view their attributes, or you can click on a specific one in order to highlight it.


Interactive tools - Right panel

Select your file from the dropdown list. To visualize the selected dataset please hit the 'Update Graph' button on the left.


Select your desired animation speed and then please hit the "Play Animation" button. You can change it by pausing and selecting a different value. You can also choose a different time interval from the interactive calendar located on the left panel.


Link to the Adjacency matrix

You can also view the filtered data set in its adjacency matrix form by navigating to Visualisations -> Vis2. Note however that in animation mode, navigating back to Vis 1 page will reset the graph!


Attention! For large datasets, it may take a while before the graph is loaded!
